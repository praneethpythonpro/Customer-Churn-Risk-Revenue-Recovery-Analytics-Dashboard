# 📊 Customer Churn Risk & Revenue Recovery Analytics

A complete business analytics project that helps companies identify customers likely to churn and estimate future revenue impact.

---

## 🚀 Features

- Data Cleaning
- Exploratory Data Analysis
- Customer Segmentation
- Revenue Trend Analysis
- Churn Risk Detection
- Revenue Forecasting
- Interactive Dashboard

---

## 📈 Technologies

- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- Scikit-Learn

---

## 📊 Dashboard

The dashboard displays

- Monthly Revenue
- Top Customers
- Customer Segments
- Churn Risk
- Revenue Forecast

---

## 📁 Dataset

Synthetic business transaction data containing

- Customer ID
- Purchase Date
- Order Value
- Region
- Product Category
- Payment Method

---

## ⚡ Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run dashboard.py
```

---

## 📌 Business Questions Answered

- Which customers are most valuable?
- Which customers are at risk of leaving?
- Which region generates maximum revenue?
- Which product category performs best?
- How much revenue is expected next month?
